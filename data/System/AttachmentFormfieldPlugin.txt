---++ Usage
Use type =attachment= in a form.
If old attachments should be deleted add "removeold" to the 'values' column. To render an Attachment as image add "image".
You can define the size of an image by adding it to the size-column (eg '200x200'), otherwise the width and height is set to 'auto'.

Example:

| AttachmentImage | attachment  | 300x300 | removeold  |  |  |
